# My personal website

This is the code of my personal website, which is powered by hugo.

## Develop

```
$ git clone git@github.com:vgjm/website.git
$ cd website
$ hugo server -D
```

## Deploy

```
$ git clone git@github.com:vgjm/vgjm.github.io.git ../vgjm.github.io
$ hugo -D -d ../vgjm.github.io
```
